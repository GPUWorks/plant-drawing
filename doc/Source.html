bool initializeOpenGL()

Intialises OpenGL, creates a window, setting the the window resize callbacks, keypress callbacks, aspect ratio, window size limits and loads OpenGL function pointers using GLAD. Returns true unless the initialisation of any component fails.


|| Main

int main()

Initialises OpenGL, creates a new Scene object to pre-process the vectors to be rendered, draws the background and the trees as long as window is open. Upon closing, deallocates all resources via Scene's destructor and terminates GLFW.
